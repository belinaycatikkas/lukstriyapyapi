# Seferihisar Kıyı Villaları

Modern ve profesyonel bir villa projesi tanıtım web sitesi.

## Özellikler

- ✅ Responsive tasarım (mobil, tablet, desktop)
- ✅ Modern ve profesyonel arayüz
- ✅ A Tipi ve B Tipi villa detay sayfaları
- ✅ Kat planları ve görseller
- ✅ İletişim formu
- ✅ Smooth animasyonlar ve hover efektleri

## Teknolojiler

- HTML5
- CSS3 (Modern CSS Variables, Grid, Flexbox)
- JavaScript (Vanilla JS)
- Google Fonts (Inter)

## Sayfalar

- `index.html` - Ana sayfa
- `a-tipi-villa.html` - A Tipi müstakil villa detay sayfası
- `b-tipi-villa.html` - B Tipi ikiz villa detay sayfası
- `contact.html` - İletişim sayfası
